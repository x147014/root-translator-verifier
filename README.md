# 🌱 词根翻译与校验工具（Root Translator & Verifier）

> 一款基于浏览器的 **离线 HTML 工具**，支持中文短语按词根智能翻译为英文，并对数据库表名/字段名进行命名规范校验。无需网络，数据完全本地处理，安全可靠。

## ✨ 功能亮点

- 🔤 **中文 → 英文词根翻译**  
  输入中文短语（如“客户满意度”），自动拆分为词根（客户 + 满意度），并映射为标准英文术语（`customer_satisfaction`）。
  
- ✅ **命名规范校验**  
  支持校验表名/字段名是否符合企业数据仓库（EDW）、数据集市（DM）等常见命名规范：
  - 仅允许小写字母、数字、下划线
  - 禁止连续下划线、开头/结尾下划线
  - 长度限制（默认 ≤ 64 字符）
  - 自动提示违规项与建议

- 📚 **词根库管理**  
  - 支持从 Excel 文件（`root_dictionary.xlsx`）导入自定义词根
  - 可导出当前词根库为 Excel
  - 内置常用业务词根示例（如：客户→customer，订单→order）

- 🖥️ **纯前端 · 完全离线**  
  所有逻辑在浏览器中运行，**不上传任何数据**，保护敏感信息。

## 🚀 快速开始

### 1. 下载文件

### 2. 准备词根库
将 `root_dictionary.xlsx` 与 `RootTransVerTool.html` 放在同一文件夹中。  
> 💡 词根库格式：Excel 第一列为中文词根，第二列为对应英文（无标题行）。

### 3. 打开工具
用 **Chrome、Edge 或 Firefox** 浏览器打开 `RootTransVerTool.html`，即可使用！

## 🛠 使用说明

### 翻译模式
1. 在左侧输入中文短语（如“产品退货率”）
2. 点击 **翻译**，右侧显示推荐英文名（如 `product_return_rate`）
3. 若词根未匹配，会高亮提示，可手动补充词根库

### 校验模式
1. 切换到 **校验** 标签页
2. 输入待校验的英文名（如 `User_ID_`）
3. 点击 **校验**，系统将指出问题（如“结尾不能有下划线”）

### 词根管理
- **导入**：选择本地 `root_dictionary.xlsx` 文件加载词根
- **导出**：将当前词根保存为新 Excel 文件
- **重置**：恢复内置默认词根


## ⚠️ 注意事项

- 由于浏览器安全策略，**必须通过 `file://` 本地打开 HTML 文件**，不能直接拖进浏览器标签页（部分浏览器会阻止读取 Excel）。
- 推荐使用 **Chrome** 或 **Edge**，Firefox 对本地文件读取限制较严。
- 词根库需为 `.xlsx` 格式（非 `.xls`），且**不要包含表头**。
